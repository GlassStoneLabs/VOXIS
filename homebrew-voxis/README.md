# Homebrew Tap — Voxis 4.0 DENSE

Professional audio restoration powered by Trinity V8.1 — Glass Stone LLC

## Install

```bash
brew tap GlassStoneLabs/voxis
brew install --cask voxis
```

## Requirements

- macOS 12+ (Monterey or later)
- Apple Silicon (arm64)
- FFmpeg (`brew install ffmpeg`)

## Upgrade

```bash
brew upgrade --cask voxis
```

## Uninstall

```bash
brew uninstall --cask voxis
```
